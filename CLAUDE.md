# Project: バナープロンプトメーカー（GPT Image 2 用）

## ✨ やること（目的）
- GPT Image 2（OpenAIの画像生成モデル）で**広告バナー画像**を生成するための、構造化プロンプトを作成するWebツール。
- フォームに入力するだけで、JSON形式・英語プロンプト両対応のプロンプトが自動生成される。
- 非エンジニアでも、プロ品質のバナー用プロンプトを誰でも簡単に作れることがゴール。

## 🧱 構成・技術スタック
- **単一HTMLファイル**（`index.html`）で完結。フレームワーク・ビルドプロセス不要。
- Vanilla HTML / CSS / JavaScript のみ使用。
- フロントエンド完結型のため、サーバー不要・APIキー不要。

## 📁 主要ファイル
- `index.html` — ツール本体（UI・ロジック・スタイル全て含む）
- `.gitignore` — `.claude/settings.local.json`, `.DS_Store`, `.env*`, 鍵ファイルを除外
- `.claude/` — Claude Code 用設定ディレクトリ

## 🌐 デプロイ・公開情報
- **公開URL**: https://banner-prompt-tool.netlify.app
- **GitHubリポジトリ**: https://github.com/fantastica077-afk/banner-prompt-maker
- **デプロイ方法**: Netlify × GitHub 連携（`main` ブランチへの push で自動デプロイ）
- **更新の流れ**: ローカル編集 → `git commit` → `git push origin main` → 1〜2分後に本番反映

## 🎨 実装済み機能（概要）
- テーマ・ジャンル選択（12種類＋自由入力）
- 商品名・ターゲット性別・ターゲット年代
- サイズ選択（SNS・LINEリッチメニュー等のプリセット＋自由入力px）
- メインビジュアル／背景・雰囲気
- メインコピー／サブコピー／USP（最大6つ）
- 価格表示（通常・特別）・CTA
- バッジ（最大3つ）・権威性エンブレム（最大3つ）
- **🎲 おまかせボタン**：各自由記述欄で、テーマに合ったサンプル文を自動入力
- 出力形式：JSON形式 / 英語プロンプト / 両方を切り替え可能
- ワンクリックコピー、シンタックスハイライト付きJSON表示

## 🤝 ルール（AIへの行動方針）
- 全て日本語で回答すること。非エンジニアでも分かるように専門用語は極力使わない、もしくは噛み砕くこと。
- プラン変更時や破壊的操作（ファイルの削除・大量上書き、リポジトリ削除など）の実行前は、必ず「実行してよいか（y/n）」を尋ねること。
- 文体は親しみやすく、かつ丁寧に統一すること。
- コードを生成・修正した際は、何が変わったのか要点を解説すること。
- 修正後は `git commit` → `git push` して本番サイトに反映することも想定する（ユーザーから「デプロイして」と依頼された場合）。

## 🎯 ターゲット・前提条件
- 環境: macOS（ユーザー: oyasho）
